# Canvas pour les TP symfony

## Prérequis
### Minimal
* `php >= 7.3`
* Postman (ou curl pour les téméraires)
### Conseillé 
* `docker`
* `docker-compose`
## Lancement

Si vous avez docker 

`docker-compose up`

Sinon

`cd canvas/public; php -S localhost:8000`

Le projet répond à cette url `http://localhost:8000/`
